当前结果：单网格数量、总持仓数量、网格均价、总名义价值、估算杠杆、每日/每月收益。

单网格数量（币）
-
单网格数量 = 总持仓数量 / 网格数量
总持仓数量（币）
-
总持仓数量 = 总名义价值 / 网格均价
网格均价（USDT）
-
等差：均价 = (区间下限 + 区间上限) / 2；等比：按几何级数价格求平均
总名义价值（USDT）
-
总名义价值 = 投资金额 × 倍数 × (1 - 预留率%)
估算杠杆倍数
-
估算杠杆 = 总名义价值 / 投资金额
预估爆仓价（USDT）
-
预估爆仓价(简化) = 网格均价 - 投资金额 / 总持仓数量
每日收益（USDT）
-
每日收益 = 单格收益 × 每天成交网格；收益率 = 每日收益 / 投资金额
每月收益（USDT）
-
每月收益 = 每日收益 × 30；月收益率 = 每月收益 / 投资金额
